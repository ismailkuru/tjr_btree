Module Base_types.Base_types_params

module Base_types_params: Bt_params

The code is heavily parameterized. We don't want function arguments to be too numerous. So we typically have a single "parameters" object ps which is more-or-less untyped. To ensure a parameter is always accessed using a particular name, and to enforce that the value of the parameter has a given type, we define "parameter accessor" functions below.
