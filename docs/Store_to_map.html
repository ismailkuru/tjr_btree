Module Store_to_map

module Store_to_map: sig .. end
The essential B-tree functionality: implement a map on top of a store.
type 't page_ref_ops = (Page_ref_int.page_ref, 't) Base_types.mref
The B-tree code exports a pre_map_ops version of a map, with explicit passing of the reference to the root of the B-tree. In order to implement the map_ops interface, we need to store the "current" reference to the B-tree root in the global state somehow. A value of type 't page_ref_ops reveals how to read and write this reference in the global state.
